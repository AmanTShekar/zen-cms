# Changelog

All notable changes to the Zenith Nucleus will be documented in this file.

---

## [0.2.0-beta] - 2026-05-13

### Added
- **High-Fidelity Documentation**: Complete suite of architectural, security, and installation guides.
- **Industrial Blueprints**: New starter templates for Next.js, Ecommerce, and Docs portals.
- **Neural Bridge**: Advanced webhook engine with HMAC security and auditing.
- **Unified Adapter Layer**: Foundation for the upcoming SQL Catalyst (PostgreSQL).

### Changed
- **Architectural Hardening**: Strictly enforced monorepo boundaries via dependency-cruiser.
- **Lazy-Loaded Engine**: Refactored the core router factory for millisecond cold starts.
- **Clean Industrial Aesthetic**: Overhauled the Admin UI and documentation for a professional, no-emoji look.

### Fixed
- **Monorepo Circularity**: Resolved illegal imports between core and admin packages.
- **Validation Bottlenecks**: Optimized Zod schema generation for high-throughput environments.

---

## [0.1.0-alpha] - 2026-05-11

### Added
- **Initial Nucleus**: Core Express engine and MongoDB adapter.
- **Spatial Editor**: First-generation 2D canvas for content orchestration.
- **Basic Admin UI**: Initial dashboard for collection management.
